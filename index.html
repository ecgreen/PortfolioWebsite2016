Design Fundamentals
Design is far more than just the look of an application. When I redesigned this web app, I updated it with modern styling, but I put even more effort into reworking user interactions through animations to make it feel simple and natural to use.
Custom User Experiences
User interfaces should be intuitive, but they should also be unique. In this application most of the functionality was on a single tab. I created a custom bar with a raised and rounded center button so the user is directed to the most important page of the application - exploring user generated geolocations.
Unique Designs
Sometimes a simple UI element can be extended to do far more in an application that is typical. While building this iPhone application, I saw an opportunity to make the calendar the center piece of user interaction. The calendar displays what days a test was completed by circling the date, and gives an idea of the result based on the color of the circle.
